# stack

A [stack]("https://en.wikipedia.org/wiki/Stack_(abstract_data_type)") is a **LIFO** (last in, first out) data structure.

## Complexity

| Category        | Complexity |
|-----------------|------------|
| Search time     | N/A        |
| Space           | $O(n)$     |
| Insert          | $O(1)$     |
| Removal         | $O(1)$, only pops |

## Uses
Basis of recursive routines and architecture.
