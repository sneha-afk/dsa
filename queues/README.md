# queue

A [queue](https://en.wikipedia.org/wiki/Queue_(abstract_data_type)) is a **FIFO** (first in, first out) data structure.

## Complexity

| Category        | Complexity |
|-----------------|------------|
| Search time     | N/A        |
| Space           | $O(n)$     |
| Insert          | $O(1)$     |
| Removal         | $O(1)$, only dequeues |

## Uses
Buffers, breadth first search.
