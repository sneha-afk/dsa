# trie

A [trie](https://en.wikipedia.org/wiki/Trie) is "a form of string-indexed look-up data structure." 

## Complexity

| Category | Complexity |
|----------|------------|
| Time     | $O(n)$     |
| Space    | $O(n)$     |
| Insert   | $O(n)$     |


## Uses
* Efficient for storing words and prefixes as each node determines a new pathway for words/strings to form
